# MyOshiCSS static status page

This branch converts the repo into a minimal static site for GitHub Pages.

It serves a single message explaining that `myoshicss.co` is temporarily down due to an issue with the MyOshi API.

## Files used by GitHub Pages

- `index.html` - main status page
- `404.html` - same content, so unknown routes still show the downtime message
- `styles.css` - minimal styling
- `CNAME` - custom domain for GitHub Pages

## Recommended Pages config

Deploy from this branch (`mouse/static-gh-pages-down-message-20260419`) and the repository root.
